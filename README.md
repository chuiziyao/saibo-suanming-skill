## ✨ 功能特性

| 功能模块 | 说明 |
|---------|------|
| 🔮 四柱八字排盘 | 输入公历生日+时辰+性别+出生地，自动排盘年柱/月柱/日柱/时柱 |
| 📈 大运推算 | 8步大运，自动判断顺排/逆排，计算起运岁数 |
| 🔍 过去回顾 | 根据大运历史，回顾过去关键年份事件 |
| 🔮 未来预测 | 逐年流年预测（5-10年），分析吉凶趋势 |
| 💰 财运预测 | 正财/偏财分析，流年财运走势 |
| 👔 官运/事业预测 | 正官/七杀分析，事业起伏趋势 |
| 💕 婚姻/感情预测 | 正财/正官分析，桃花运、感情走势 |
| 👶 子女运预测 | 官杀/食伤分析，子女缘分与未来发展 |
| 🏥 健康预测 | 五行平衡分析，流年健康注意事项 |
| 🎯 趋吉避凶建议 | 喜用神分析，幸运颜色/方位/数字/行业建议 |
| ✨ 神煞查询 | 天乙贵人、文昌、驿马、桃花、华盖等 |
| 📊 格局判断 | 正官格、七杀格、正财格、偏财格、食神格、伤官格等 |

## 🚀 安装方法

### 方法一：从 GitHub 安装（推荐）

```bash
# WorkBuddy CLI 安装（如果支持）
workbuddy skill install https://github.com/chuiziyao/saibo-suanming-skill

# 或者手动下载
git clone https://github.com/chuiziyao/saibo-suanming-skill.git ~/.workbuddy/skills/saibo-suanming
```

### 方法二：手动安装

1. 下载本仓库 ZIP 文件
2. 解压到 `~/.workbuddy/skills/saibo-suanming/` 目录
3. 重启 WorkBuddy 客户端

## 📖 使用方法

安装完成后，在 WorkBuddy 对话中直接说：

- 「帮我算命」
- 「看八字」
- 「赛博算命」
- 「测财运/测事业/测婚姻」

AI 会引导您输入：
1. **出生年月日**（公历）
2. **出生时辰**（精确到小时，时辰对八字影响巨大）
3. **性别**（男/女）
4. **出生地点**（用于时区参考）

## 🔧 技术实现

- **无外部依赖**：纯 Python 标准库实现
- **排盘算法**：
  - 五虎遁年起月法
  - 五鼠遁日起时法
  - 男命阳年/女命阴年顺排大运
  - 男命阴年/女命阳年逆排大运
  - 起运岁数 = 出生日到节气天数 ÷ 3
- **命理规则**：
  - 天干地支、五行相生相克
  - 十神计算（正官、七杀、正财、偏财、食神、伤官等）
  - 神煞查询（天乙贵人、文昌、驿马、桃花、华盖）
  - 十二长生状态
  - 格局判断
  - 喜用神分析
