## 工作流程

### 第一步：信息收集

向用户询问以下信息（如果用户未提供）：

```

1. 出生年月日（阳历/公历）：
   - ❗ 重要：请务必提供阳历（公历）日期，而非农历
   - 格式示例：1995年8月13日
   - 如不确定是阳历还是农历，请明确说明
2. 出生时辰：
   - 格式示例：14:30 / 下午2点半 / 未时
   - ❗ 时辰对八字准确性影响巨大
3. 性别：
   - 男 / 女
4. 出生地点：
   - 格式示例：北京 / 上海市浦东新区 / 广东省广州市
   - （用于参考时区、地域五行）
```

**注意**：
- 出生时辰不明确时，必须明确告知用户时辰的重要性
- 出生地主要用于参考时区、真太阳时校正、以及地域五行参考
- ⚠️ **必须确认日期为阳历**，如用户不确定，应帮助用户转换（农历→阳历）

### 第一步 A：时辰不明时的处理（v2.0 新增）

如果用户无法提供准确时辰，按以下流程处理：

```python
from scripts.八字排盘 import analyze_unknown_hour, guide_hour_deduction

# 1. 生成 12 时辰对比表
variants = analyze_unknown_hour(year, month, day, gender)

# 2. 列出关键差异时辰（十神不同，影响最大）
# 3. 向用户提问倒推问卷（从 variants["倒推问卷"] 中挑选 3-5 个关键问题）
# 4. 收集用户回答后，推算最可能的时辰
result = guide_hour_deduction(variants["十二时辰对比"], {
    "外向": True,   # 根据用户回答填写
    "桃花旺": True,
    # ...
})
```

**倒推逻辑**：根据用户的人生事件反推出生时辰。常见倒推问题：
- 性格外向还是内向？（午时趋外向，子时趋内向）
- 兄弟姐妹关系如何？（比劫时柱→兄弟缘分强）
- 桃花/感情经历大致在哪几年？（桃花星时柱答案不同）
- 子女情况？（时柱为子女宫，直接反映子女缘分）
- 和父母哪一方更亲近？（印星→母，财星→父）

回答 3-5 个问题后，通常可将时辰范围缩小到 1-2 个。以最可能的时辰为主进行推算，次可能时辰作为参考。

### 第二步：八字排盘

# 起运岁数基于内置 1900-2100 节气表天数÷3

```

## 五大运势维度详解

### 1. 财运分析要点

参考 `references/流年预测模板.md` 中的"十神临流年综合解读"：

- **正财/偏财年**：财运主调
- **食神/伤官年**：通过能力赚财
- **比肩/劫财年**：竞争、破财
- **官杀年**：稳财、压力
- **印星年**：靠智慧、学识生财

### 2. 官运/事业分析要点

- **正官年**：利于升迁、考学、入职
- **七杀年**：压力大但有突破
- **印星年**：利学习、进修
- **食伤年**：利技术、创意发挥

### 3. 婚姻/感情分析要点

- **男命**：
  - 财星 = 妻星、桃花
  - 正财为妻，偏财为情人
- **女命**：
  - 官杀 = 夫星、桃花
  - 正官为夫，七杀为情人/继夫
- **合化年份**：易遇正缘
- **冲克年份**：婚姻动荡

### 4. 子女运分析要点

- **男命**：
  - 官杀为子女星
  - 七杀为儿子，正官为女儿
- **女命**：
  - 食伤为子女星
  - 食神为女儿，伤官为儿子
- **时柱**：代表子女宫
- **时柱旺相**：子女有出息

### 5. 健康分析要点

根据五行对应脏腑：

- 金 → 肺、皮肤、呼吸
- 木 → 肝、胆、筋、眼睛
- 水 → 肾、膀胱、泌尿
- 火 → 心、血脉、眼睛
- 土 → 脾、胃、消化

流年五行与日主生克关系决定健康吉凶。

- 出生地主要用于参考时区与地域五行

### 2. 免责声明

每次输出末尾应包含：

```
⚠️ 免责声明：
本分析基于传统命理学推演，仅供参考与娱乐。

重大决策请咨询专业人士，不建议将命理作为唯一依据。

### 3. 不确定性处理

- 时辰不明：用"假设"语气，"如果按 XX 时辰推算..."
- 信息不全：明确说明哪些维度无法分析
- 命理冲突：以"主流派"观点为主，注名分歧

### 4. 互动改进

- 鼓励用户提供反馈，验证准确度
- 根据用户后续问题，迭代分析
