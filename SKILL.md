# 赛博算命

> **作者：锤无双** | 详细使用说明请关注 B 站同名「锤无双」

## 概述

本技能基于中国传统命理学（四柱八字、子平命学），结合现代软件工程实现。用户输入公历出生年月日时、性别、出生地后，将获得：

1. 完整的四柱八字排盘
2. 大运推算（每 10 年一步）+ **每步大运关键事件预测**
3. **回顾过去**：从出生至今的大运流年关键事件还原
4. **展望未来**：逐年（流年）运势预测
5. 五大运势维度：财运、官运/事业、婚姻/感情、子女运、健康
6. 各步大运关键事件：学业/升学、事业转折、婚姻大事、子女出生、父母健康、官司纠纷、意外伤病、迁移搬家
7. 趋吉避凶的具体建议

## 触发条件

当用户消息中包含以下内容时，触发本技能：

- "算命"、"看八字"、"排八字"、"算流年"
- "赛博算命"、"命理分析"、"测运势"
- "财运"、"官运"、"婚姻运"、"子女运"
- "看命"、"查命格"、"八字合婚"
- 任何要求基于生辰进行命运推演的需求

## 工作流程

### 第一步：信息收集

向用户询问以下信息（如果用户未提供）：

```
🔮 赛博算命 · 信息收集

1. 出生年月日（阳历/公历）：
   - ❗ 重要：请务必提供阳历（公历）日期，而非农历
   - 格式示例：1995年8月13日
   - 如不确定是阳历还是农历，请明确说明
2. 出生时辰：
   - 格式示例：14:30 / 下午2点半 / 未时
   - ❗ 时辰对八字准确性影响巨大
3. 性别：
   - 男 / 女
4. 出生地点：
   - 格式示例：北京 / 上海市浦东新区 / 广东省广州市
   - （用于参考时区、地域五行）
```

**注意**：
- 出生时辰不明确时，必须明确询问
- 出生地主要用于参考时区、真太阳时校正、以及地域五行参考
- ⚠️ **必须确认日期为阳历**，如用户不确定，应帮助用户转换（农历→阳历）

### 第二步：八字排盘

调用 `scripts/八字排盘.py` 中的 `calc_bazi()` 函数进行排盘：

```python
from scripts.八字排盘 import calc_bazi, calc_dayun, get_shensha, get_geju, analyze_xiyongshen, predict_life_events

bazi = calc_bazi(year, month, day, hour, gender)
# bazi 包含：年柱、月柱、日柱、时柱、日主、五行统计

dayun = calc_dayun(bazi, gender)
# dayun 包含：起运岁数、大运方向、8 步大运

shensha = get_shensha(bazi['day_pillar'], bazi['hour_pillar'], bazi['year_pillar'])
geju = get_geju(bazi)
xiyong = analyze_xiyongshen(bazi)

# 大运关键事件预测
life_events = predict_life_events(bazi, dayun, gender)
# life_events 包含：每步大运的学业、事业、婚姻、子女、健康等关键事件
```

## 五大运势维度详解

### 1. 财运分析要点

参考 `references/流年预测模板.md` 中的"十神临流年综合解读"：

- **正财/偏财年**：财运主调
- **食神/伤官年**：通过能力赚财
- **比肩/劫财年**：竞争、破财
- **官杀年**：稳财、压力
- **印星年**：靠智慧、学识生财

### 2. 官运/事业分析要点

- **正官年**：利于升迁、考学、入职
- **七杀年**：压力大但有突破
- **印星年**：利学习、进修
- **食伤年**：利技术、创意发挥

### 3. 婚姻/感情分析要点

- **男命**：
  - 财星 = 妻星、桃花
  - 正财为妻，偏财为情人
- **女命**：
  - 官杀 = 夫星、桃花
  - 正官为夫，七杀为情人/继夫
- **合化年份**：易遇正缘
- **冲克年份**：婚姻动荡

### 4. 子女运分析要点

- **男命**：
  - 官杀为子女星
  - 七杀为儿子，正官为女儿
- **女命**：
  - 食伤为子女星
  - 食神为女儿，伤官为儿子
- **时柱**：代表子女宫
- **时柱旺相**：子女有出息

### 5. 健康分析要点

根据五行对应脏腑：

- 金 → 肺、皮肤、呼吸
- 木 → 肝、胆、筋、眼睛
- 水 → 肾、膀胱、泌尿
- 火 → 心、血脉、眼睛
- 土 → 脾、胃、消化

流年五行与日主生克关系决定健康吉凶。

## 重要原则

### 1. 信息确认

- 时辰对命盘准确性影响最大，**不明确必须追问**
- 出生地主要用于参考时区与地域五行

### 2. 免责声明

每次输出末尾应包含：

```
⚠️ 免责声明：
本分析基于传统命理学推演，仅供参考与娱乐。
命由天定，运在人为；积极心态、不断努力才是改变命运的根本。
重大决策请咨询专业人士，不建议将命理作为唯一依据。
```

### 3. 不确定性处理

- 时辰不明：用"假设"语气，"如果按 XX 时辰推算..."
- 信息不全：明确说明哪些维度无法分析
- 命理冲突：以"主流派"观点为主，注名分歧

### 4. 互动改进

- 鼓励用户提供反馈，验证准确度
- 根据用户后续问题，迭代分析
- 可深入到具体月份、流日、流时
