# 赛博算命 Skill - 安装指南

## 前置依赖

```bash
# v2.0 必须安装 sxtwl（寿星天文历）
pip install sxtwl
```

## 📥 安装方法

### Git Clone

```bash
git clone https://github.com/chuiziyao/saibo-suanming-skill.git ./skills/saibo-suanming
```

### 下载 ZIP

1. 访问 https://github.com/chuiziyao/saibo-suanming-skill
2. 点击绿色「Code」→「Download ZIP」
3. 解压到 AI 智能体平台的 `skills/saibo-suanming/` 目录

## ⚠️ 常见问题

### 1. 安装后 skill 不显示？

**答**：确保解压到了正确的 skills 目录。各平台 skills 目录位置可能不同，请参考对应平台的文档。

### 2. Python 脚本无法运行？

**答**：v2.0 需要安装 sxtwl：`pip install sxtwl`。如果 sxtwl 安装失败，脚本会自动降级为简化算法（精度较低）。Python 版本需 ≥ 3.8。

### 3. 中文文件名报错？

**答**：使用 importlib 加载：

```python
import importlib.util
spec = importlib.util.spec_from_file_location('m', 'scripts/八字排盘.py')
m = importlib.util.module_from_spec(spec)
spec.loader.exec_module(m)
```
