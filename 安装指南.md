# 赛博算命 Skill - 安装指南

## 📥 方法一：Git Clone（推荐）

```bash
# WorkBuddy 用户
git clone https://github.com/chuiziyao/saibo-suanming-skill.git ~/.workbuddy/skills/saibo-suanming

# OpenClaw 用户
git clone https://github.com/chuiziyao/saibo-suanming-skill.git ~/.openclaw/skills/saibo-suanming
```

---

## 📥 方法二：下载 ZIP

1. 访问 https://github.com/chuiziyao/saibo-suanming-skill
2. 点击绿色「Code」按钮
3. 选择「Download ZIP」
4. 解压到 skills 目录：
   - WorkBuddy：`~/.workbuddy/skills/saibo-suanming/`
   - OpenClaw：`~/.openclaw/skills/saibo-suanming/`

---

## 📥 方法三：ClawHub（OpenClaw 用户）

```bash
npm i -g clawhub
clawhub login
clawhub skill install chuiziyao/saibo-suanming-skill
```

---

## ✅ 验证安装

安装完成后，重启 WorkBuddy/OpenClaw 客户端，然后说：

## ⚠️ 常见问题

### 1. 安装后 skill 不显示？

**答**：重启客户端。Skills 目录是客户端启动时扫描的。

### 2. Python 脚本无法运行？

**答**：本 skill 使用纯 Python 标准库，无需安装额外依赖。确保您的 Python 版本 ≥ 3.8。

### 3. 中文文件名报错？

**答**：确保您的 Python 脚本执行方式正确：

```python
import importlib.util
spec = importlib.util.spec_from_file_location('m', 'scripts/八字排盘.py')
m = importlib.util.module_from_spec(spec)
spec.loader.exec_module(m)
```
